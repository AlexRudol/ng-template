# ng-seed

Angular Webpack seed project.

## Features

- [Angular 4](https://angular.io/) (Typescript)
- [Webpack 3](https://webpack.js.org/)
- Ahead-of-Time, tree-shaking, compression (~95kb)
- Easy configurable dev-server (protocol, headers etc)

## Prerequirements

`node`, `npm`

## Set up


```
    npm i
```

## Run


```
    npm start
```


```
    npm run build
```
